## 🌟 Overview
**HisabKhata** is a high-fidelity digital ledger designed to help small businesses and individuals manage their accounts with ease. Inspired by modern accounting tools, it provides a clean, mobile-first interface to track "Give" and "Get" transactions, manage customer relationships, and automate financial communication.

## ✨ Key Features
- 👥 **Customer Management**: Effortlessly organize and track individual customer profiles.
- 💸 **Transaction Tracking**: Log credits and debits with real-time balance calculations.
- 📧 **Automated Alerts**: Integrated notifications for transactions and welcome messages.
- 📊 **Visual Insights**: Interactive charts and data visualization using Recharts.
- 📱 **Responsive Design**: Pixel-perfect UI optimized for both mobile and desktop.
- 🔒 **Secure Auth**: Robust user authentication powered by Firebase.

## 🚀 Getting Started

### Prerequisites
- Node.js (v18+)
- npm or yarn
- Firebase CLI (`npm install -g firebase-tools`)

### Installation
1. **Clone the repository:**
   ```bash
   git clone https://github.com/yourusername/HisabKhata.git
   cd HisabKhata
   ```

2. **Install all dependencies:**
   ```bash
   npm run install:all
   ```

3. **Set up Environment Variables:**
   Create a `.env` file in the `accounts/` directory with your Firebase configuration.

### Local Development
To start the development server for the frontend:
```bash
npm run dev
```
